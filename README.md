# pythonlearn
自学《Python基础教程》学习笔记

跟着书上的一步一步在解释器Pycharm上记下自己觉得重要的笔记
## learnPDF
learnPDF主要讲述了Python的一些基础知识，当然，如果学过C，这部分其实并不是多难
主要内容包括：
（1）数和表达式
（2）变量
（3）语句
（4）获取用户输入
（5）函数
（6）模块
（7）保存并执行程序
（8）字符串
（9）小结
## learnPDF2
learnPDF2主要讲述两个数据类型：列表和元组，属于Python中比较重要的概念
（1）序列概述
（2）通用的序列操作
（3）列表：Python的主力
（4）元组：不可修改的序列
（5）小结
## learnPDF3

（1）字符串的基本操作
（2）设置字符串的格式：详细
     替换字段名
     基本转换
     宽度，精度和千位分隔符
     符号，对齐和用0填充
（3）字符串方法
      center
      find
      join
      lower

      strip
      translate
 (4)判断字符串是否满足特定条件
（5）小结
